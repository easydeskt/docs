---
title: Команды бота
description: Полный справочник команд Telegram-бота для операторов
---

:::note[Раздел в разработке]
Полное описание каждой команды с примерами готовится.
:::

Все команды вводятся в топике тикета внутри Telegram-суперагруппы.

| Команда | Описание |
|---------|----------|
| `/assign me` | Назначить тикет на себя |
| `/assign @username` | Назначить тикет на другого агента |
| `/free` | Снять назначение (освободить тикет) |
| `/resolve` | Отметить тикет как решённый |
| `/close` | Закрыть тикет без решения |
| `/merge #id` | Объединить с другим тикетом |
| `/priority low\|normal\|high\|urgent` | Установить приоритет |
| `/reply Текст` | Отправить ответ клиенту |
| `/note Текст` | Добавить внутреннюю заметку (не видна клиенту) |
| `/notes` | Посмотреть все заметки о клиенте |

:::tip
`/assign me` работает даже если тикет уже назначен на кого-то другого — это **перехват**. Система не блокирует передачу; достаточно договориться с коллегой.
:::
