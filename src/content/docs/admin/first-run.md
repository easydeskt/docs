# Первый запуск

:::note[Раздел в разработке]
Полная инструкция с примером `.env` готовится.
:::

## Схема первоначальной настройки

1. Развернуть EasyDesk через Docker Compose (см. [Деплой](/docs/dev/deployment/))
2. Перед запуском задать переменную `BOOTSTRAP_ADMIN_TG_ID` в `.env` — Telegram User ID первого администратора
3. При первом старте система создаёт учётную запись администратора автоматически
4. Открыть Mini App через бота суперагруппы и войти под своим аккаунтом
5. Подключить каналы, добавить агентов, создать теги и шаблоны

:::tip
Узнать свой Telegram User ID можно через бота [@userinfobot](https://t.me/userinfobot).
:::

:::caution
`BOOTSTRAP_ADMIN_TG_ID` используется только один раз при первом запуске. После создания администратора эту переменную можно убрать из `.env`.
:::
