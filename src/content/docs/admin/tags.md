# Теги и цвета

:::note[Раздел в разработке]
Полное описание с иллюстрациями интерфейса готовится.
:::

Теги — метки для категоризации тикетов. Управляются в разделе **Администрирование → Теги** в Mini App.

## Что можно сделать

- Создать тег с произвольным именем
- Задать цвет тега (шестнадцатеричный код или пикер в интерфейсе)
- Переименовать или удалить тег

## Как операторы используют теги

Оператор назначает теги через карточку тикета в Mini App. Один тикет может иметь несколько тегов. В списке тикетов можно фильтровать по тегам.

:::tip
Заранее договоритесь с командой о структуре тегов. Хаотичный набор тегов быстро теряет смысл.
:::
