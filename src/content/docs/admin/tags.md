title: Теги и цвета
description: Создание и настройка тегов для категоризации тикетов
---

Теги — метки для категоризации тикетов. Управляются в разделе **Администрирование → Теги** в Mini App.

## Создание тега

1. Открыть Mini App → **Теги** → **Создать**.
2. Ввести название тега (латиница или кириллица, без пробелов лучше использовать дефис).
3. Выбрать цвет через пикер или ввести hex-код.

Цвет тега отображается в списке тикетов Mini App рядом с его названием — помогает быстро сканировать список визуально.

## Редактирование и удаление

Название и цвет тега можно изменить в любой момент. Изменения применяются ко всем тикетам, где этот тег уже назначен.

При удалении тега он снимается со всех тикетов автоматически. Тикеты при этом не удаляются.

## Советы по структуре тегов

Хаотичный набор тегов быстро теряет смысл. Несколько принципов, которые работают:

- Используйте теги для **типа обращения**, а не для статуса — статус и так есть у каждого тикета.
- Договоритесь о нескольких базовых категориях до начала работы: `billing`, `bug`, `question`, `feature-request`.
- Добавляйте специальные теги по мере необходимости: `vip`, `sla`, `refund`.
- Не создавайте теги под конкретные тикеты — это быстро превращается в мусор.
