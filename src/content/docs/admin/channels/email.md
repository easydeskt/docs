# Подключение канала Email

:::note[Раздел в разработке]
Полная инструкция с примером конфигурации готовится.
:::

## Что потребуется

- Email-адрес для входящих обращений
- Данные IMAP-сервера (хост, порт, SSL)
- Данные SMTP-сервера (хост, порт, SSL)
- Пароль или пароль приложения (для сервисов с 2FA: Gmail, Яндекс, Mail.ru)

## Как это работает

EasyDesk опрашивает ящик по IMAP через заданный интервал. Каждое новое письмо от клиента создаёт тикет или добавляет сообщение в существующий (если письмо в том же треде).

Ответы оператора отправляются через SMTP в том же треде — с правильными заголовками `In-Reply-To` и `References` (RFC 5322), чтобы почтовые клиенты клиента видели цельную переписку.

## Поддерживаемые вложения

Изображения, документы, PDF — всё, что может содержать письмо.
