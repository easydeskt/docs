# Подключение канала Telegram

:::note[Раздел в разработке]
Полная инструкция с примером конфигурации готовится.
:::

## Что потребуется

- Telegram-бот, созданный через [@BotFather](https://t.me/BotFather)
- Токен бота (`BOT_TOKEN`)

## Режимы работы

| Режим | Описание |
|-------|----------|
| Long Polling | Бот сам опрашивает серверы Telegram. Не требует публичного HTTPS. |
| Webhook | Telegram доставляет обновления на ваш URL. Требует HTTPS и публичного домена. |

EasyDesk поддерживает оба режима и автоматически переключается при сбоях (failover с Long Poll → Webhook и обратно).

## Поддерживаемые типы вложений

`audio` · `document` · `photo` · `sticker` · `video` · `voice`
