# Подключение канала ВКонтакте

:::note[Раздел в разработке]
Полная инструкция готовится.
:::

## Что потребуется

- Сообщество ВКонтакте с включёнными личными сообщениями
- Токен сообщества с правами на `messages`

## Режимы работы

| Режим | Описание |
|-------|----------|
| Callback API | VK отправляет события POST-запросами на ваш URL. Требует HTTPS. |
| Long Poll | Сервер сам опрашивает VK API. Работает без публичного домена. |

:::tip
Для локальной разработки и тестирования удобно использовать Long Poll — не нужен публичный домен.
:::
