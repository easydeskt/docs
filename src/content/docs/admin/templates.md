Шаблоны управляются в разделе **Администрирование → Шаблоны** в Mini App. Готовые шаблоны доступны всем операторам при работе с тикетами.

## Создание шаблона

1. Название — отображается в списке при выборе оператором
2. Текст с плейсхолдерами (опционально)
3. Вложение — изображение или документ (опционально)

## Поддерживаемые плейсхолдеры

| Плейсхолдер | Значение |
|-------------|----------|
| `{ticket_id}` | ID текущего тикета |
| `{client_name}` | Имя клиента из профиля |
| `{agent_name}` | Имя оператора, использующего шаблон |

При отправке плейсхолдеры заменяются на реальные значения в момент использования.
