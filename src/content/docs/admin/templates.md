---
title: Шаблоны с вложениями
description: Создание шаблонов быстрых ответов с поддержкой вложений
---

Шаблоны управляются в разделе **Администрирование → Шаблоны** в Mini App. Готовые шаблоны доступны всем операторам при работе с тикетами.

## Создание шаблона

1. Открыть Mini App → **Шаблоны** → **Создать**.
2. Ввести название — операторы видят его в списке при выборе.
3. Написать текст ответа с плейсхолдерами (при необходимости).
4. Прикрепить вложение — изображение или документ (при необходимости).

Шаблон может содержать только вложение без текста, только текст без вложения или и то, и другое.

## Плейсхолдеры

| Плейсхолдер | Что подставляется |
|-------------|-------------------|
| `{ticket_id}` | ID текущего тикета |
| `{client_name}` | Имя клиента из его профиля |
| `{agent_name}` | Имя оператора, использующего шаблон |

Плейсхолдеры заменяются в момент использования шаблона оператором.

**Пример:**

```
Здравствуйте, {client_name}!

Ваше обращение #{ticket_id} зарегистрировано. Меня зовут {agent_name}, я займусь вашим вопросом.
```

## Редактирование и удаление

Название, текст и вложение шаблона можно изменить в любой момент. Изменения вступают в силу сразу — операторы увидят обновлённый шаблон при следующем использовании.

Удалённый шаблон нельзя восстановить.
